# TechAlaya AI Agent Discovery Session

A professional, modern, and highly interactive multi-step onboarding form designed for **TechAlaya** to scope custom AI Agent projects and gather requirements from clients.

## 🚀 Features

- **Futuristic UI**: Deep space dark mode with neon accents (Cyan & Violet) and fluid glassmorphism layout.
- **Multi-step Form Wizard**: 6 interactive steps dividing the 10-section discovery questionnaire.
- **Client-Side Validation**: Dynamic field verification and visual error highlighting.
- **Draft Persistence**: Auto-saves form progress in `LocalStorage` to prevent data loss.
- **Summary Accordion Review**: Aggregates entered text dynamically before final submission.
- **Formspree Backend**: Seamless serverless form capture that emails responses and lists submissions.

## 🛠️ Configuration & Deployment

1. **Backend Integration**: 
   Open `index.html` and replace `YOUR_FORMSPREE_ENDPOINT_HERE` inside the `<form>` tag with your Formspree form endpoint ID.
   
2. **Hosting**:
   - Deploy directly via **GitHub Pages** (Settings > Pages > Deploy from branch).
   - Or drag-and-drop on **Netlify Drop** / **Vercel**.
